<h2>üõ†Ô∏è Installation Steps:</h2>

<p>1. Clone the repository</p>

```

```

<p>2. Create Virtual Environment  (Optional)</p>

```

```

<p>3. Activate virtual environment:</p>

```

```

<p>4. Install dependencies:</p>

```

```

<p>5. Launch Jupyter Lab from the root:</p>

```

```

> All notebooks will use the root .venv kernel for consistent package management.
