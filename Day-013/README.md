# Day 13 Machine Learning Fundamentals

Today I learned the core ideas behind how machine learning models learn from data and generalize to unseen inputs.

---

## Key Concepts Learned

### 🔹 Learning Types
- Instance-based learning: uses stored data for prediction based on similarity (e.g., KNN)  
- Model-based learning: learns a general function from data to make predictions (e.g., Linear Regression)

---

### 🔹 Data Splitting
- Training data: used to train the model  
- Testing data: used to evaluate performance  
- Validation data: used for tuning and model selection  

---

### 🔹 Model Behavior
- Overfitting: performs well on training but poorly on test data  
- Underfitting: performs poorly on both training and test data  

---

### 🔹 Bias & Variance
- Bias: error from overly simple assumptions  
- Variance: error from too much sensitivity to training data  
- Goal: balance both for good generalization
