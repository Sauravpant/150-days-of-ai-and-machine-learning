### Day 4 – Pandas Data Cleaning & Aggregation

Today I explored **Pandas** for cleaning, preprocessing, handling outliers, encoding categorical data, and aggregating/grouping data for analysis and ML readiness.

---

#### **Key Points**

- **Handling Missing Data**: Detect, fill with mean/median, or drop columns/rows.
- **Duplicates**: Identify and remove duplicate rows.
- **Data Types**: Convert columns to appropriate types and standardize column names.
- **Categorical Data**: Encode nominal and ordinal data using one-hot or label encoding.
- **Boolean Columns**: Convert boolean values to integers for ML compatibility.
- **Outlier Handling**: Detect and clip outliers using the IQR method.
- **Saving Cleaned Data**: Export processed datasets to CSV for further use.
- **Grouping & Aggregation**: Use `groupby` to summarize data and apply multiple aggregation functions.
- **Concatenation**: Combine DataFrames row-wise or column-wise.
- **Apply & Map**: Transform data efficiently using custom functions on columns or rows.
