# Day 15 – Linear Regression Implementation

Today I implemented a simple Linear Regression model to predict height using weight as the input feature.

---

## Key Learnings

- Loaded and explored the height-weight dataset
- Visualized relationship between weight and height
- Split data into training and testing sets
- Applied standardization using `StandardScaler`
- Trained model using `LinearRegression`
- Evaluated model with MSE, MAE, RMSE, and R2 score
- Created a prediction function for new weight values
