# Day 2 – NumPy Advanced Operations

Today I explored **NumPy’s advanced features**, focusing on array broadcasting, views vs copies, boolean indexing, universal functions, linear algebra, data types, and random number generation for hands-on numerical computing.

---

## Key Points

- **Array Shapes & Reshape:** 1D & 2D arrays, `.shape`, `.reshape()`
- **Broadcasting:** Arrays of different shapes can operate together if dimensions match or are 1
- **Views vs Copies:**
  - View shares data (`.base` returns original)
  - Copy has separate data (`.base` is None)
- **Boolean Indexing:** Select/modify elements with conditions, combine masks
- **Universal Functions:** Element-wise math, trig, comparison, bitwise, floating
- **Aggregation & Stats:** `min`, `mean`, `std`, `var`, `percentile`, `average`, `histogram`
- **Linear Algebra:** Dot, matrix power, eigenvalues, solve Ax=B, inverse, diagonal
- **Data Types:** `int32`, `int64`, `float32`, `bool`, convert with `.astype()`
- **Random Module:** `rand`, `randn`, `normal`, `randint`, `choice`, `shuffle`, `permutation`, `seed()`
