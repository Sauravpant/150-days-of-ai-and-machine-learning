# Day 16 - Polynomial Regression Implementation and Pipeline Concept in Machine Learning

Today I implemented polynomial regression on synthetic quadratic data and compared it with a standard linear regression model.

---

## Key Learnings

- Generated a quadratic dataset
- Trained a basic `LinearRegression` model as a baseline
- Evaluated model performance using `r2_score`
- Applied `PolynomialFeatures` to capture non-linear patterns
- Compared quadratic and cubic polynomial regression results
- Built a reusable regression pipeline with `Pipeline`
- Visualized predictions against training and testing points
