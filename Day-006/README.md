### Day 1 – Matplotlib Basics

Today I explored **Matplotlib** for visualizing data, creating plots, and styling them for clarity and analysis.

---

#### **Key Points**

- **Figure**: The window/canvas containing all plots.
- **Axes / Subplots**: The plotting area inside the figure where data is drawn.
- **Core Plots**: Line (trend), Scatter (relationship), Bar (category comparison), Histogram (distribution).
- **Labels & Titles**: Use `xlabel`, `ylabel`, and `title` to describe axes and plot.
- **Legend**: Identify multiple plots using `legend()`.
- **Styling**: Customize lines, markers, colors, linewidth, and transparency (`alpha`).
- **OO vs Pyplot**: Use `fig, ax = plt.subplots()` for control over multiple plots.
