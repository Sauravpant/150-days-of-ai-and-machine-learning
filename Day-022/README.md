# Day 22 - Support Vector Machines (Classifier, Kernels, Regression)

Today I implemented and evaluated Support Vector Machine models for both classification and regression, explored different kernels, and improved model performance using hyperparameter tuning.

---

## Key Learnings

- Learnt about **Support Vector Machines (SVM)** for both classification and regression tasks
- Built and evaluated a **Support Vector Classifier (SVC)** on synthetic binary classification data
- Explored and compared multiple **SVM kernels** including **Linear, Polynomial, RBF, and Sigmoid**
- Understood how kernel choice affects decision boundaries for non-linearly separable data
- Implemented a **Support Vector Regressor (SVR)** on the Tips dataset
- Applied preprocessing and encoding steps for categorical features before training SVR
- Performed **Hyperparameter Tuning** for both SVC and SVR using **GridSearchCV** with cross-validation
- Tuned important SVM hyperparameters such as **C, kernel, gamma, degree, and coef0**
- Evaluated classification model performance using **Accuracy Score, Confusion Matrix, and Classification Report (Precision, Recall, F1-score)**
- Evaluated regression model performance using **R2 Score, Mean Absolute Error (MAE), and Mean Squared Error (MSE)**
