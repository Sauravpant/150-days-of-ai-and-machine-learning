# Day 20 - Naive Bayes (Gaussian, Bernoulli, Multinomial and Categorical)

Today I implemented and evaluated three variants of Naive Bayes classifiers: Gaussian Naive Bayes, Bernoulli Naive Bayes, and Categorical Naive Bayes.

---

## Key Learnings
- Learnt about the **Naive Bayes Algorithm** for binary and multi class classification and its assumption of  independece among the input features
- Learnt about the various variants of **Naive Bayes** and usecase of each variant
- Built and evaluated a **Gaussian Naive Bayes** model on the Iris dataset
- Implemented **Bernoulli Naive Bayes** on binary/boolean-style feature data
- Implemented **Categorical Naive Bayes** on categorical feature data
- Assessed model performance using:
  - Accuracy Score
  - Classification Report (Precision, Recall, F1-score)
  - Confusion Matrix
