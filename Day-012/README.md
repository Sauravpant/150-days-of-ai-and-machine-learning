# Day 12 – Feature Engineering and Exploratory Data Analysis (EDA) on Flight Price Dataset

Today I worked on feature engineering and exploratory data analysis (EDA) using a real-world flight price dataset. The goal was to clean raw data, extract useful features, and prepare it for machine learning models.

---

## Key Steps

- Extracted date, month, and year features  
- Converted time into hour and minute columns  
- Handled missing values in dataset  
- Encoded categorical variables using One-Hot Encoding  
- Applied ordinal mapping for total stops  
- Dropped unnecessary and redundant columns  
- Visualized price variation with stops  
- Prepared final dataset for modeling
