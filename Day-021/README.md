# Day 21 - K Nearest Neighbours Classification, Regression and Hyperparameter Tuning

Today I implemented and evaluated K Nearest Neighbours models for both classification and regression, and improved their performance using hyperparameter tuning.

---

## Key Learnings

- Learnt about the **K Nearest Neighbours (KNN) Algorithm** and how it works for both classification and regression tasks
- Built and evaluated a **KNN Classifier** on synthetic classification data
- Built and evaluated a **KNN Regressor** on synthetic regression data
- Performed **Hyperparameter Tuning of KNN Classifier** using GridSearchCV with cross-validation
- Performed **Hyperparameter Tuning of KNN Regressor** using GridSearchCV with cross-validation
- Tuned key KNN hyperparameters including **n_neighbors, weights, algorithm, p, and leaf_size**
- Analysed classification model performance using **Accuracy Score, Classification Report (Precision, Recall, F1-score), and Confusion Matrix**
- Analysed regression model performance using **Mean Absolute Error (MAE), Mean Squared Error (MSE), and R2 Score**
- Visualized model performance using a **Confusion Matrix Heatmap** and **True vs Predicted Scatter Plot**
