# Day 17 - Salary Prediction App

Today I Built a salary prediction app that takes employee details (age, experience, job title, education, gender) and predicts salary using Linear Regression.

---

## Key Learnings

- Loaded and cleaned salary dataset
- Performed feature encoding for categorical variables (education, gender, job title)
- Kept numeric features (age, experience) as continuous values
- Trained a Linear Regression model on preprocessed data
- Saved trained model as pickle file for reuse
- Built a form-based interface for real-time predictions
- Integrated ML model with React.js web app and FastAPI backend
